## About Us

**Montimage** is a French cybersecurity company (est. 2004, Paris) specializing in network traffic analysis, deep packet inspection, and AI-powered security solutions. We build open-source tools for network monitoring, threat detection, and protocol analysis — supporting 700+ protocols.

**Core expertise:** Network Traffic Analysis • APT Detection • AI/ML Security • 5G/6G • IoT Security • Cyber Range

## Get Involved

We welcome contributions from the security and networking community:

- **Report issues** — Found a bug? Open an issue in the relevant repository
- **Submit PRs** — Bug fixes, features, and documentation improvements welcome
- **Share feedback** — Help us improve our tools with your use cases
- **Research collaboration** — We actively partner on EU-funded cybersecurity projects

Please read each repository's `CONTRIBUTING.md` for specific guidelines.
